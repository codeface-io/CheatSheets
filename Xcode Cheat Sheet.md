# Xcode Cheat Sheet

## Keyboard Shortcuts

| Intent | Key Command |
| --- | --- |
| Build | ⌘B |
| Run | ⌘R |
| Go to previous file | ⌃⌘← |
| Go to next file | ⌃⌘→ |
| Show current file in navigator (jump to it) | ⇧⌘J |
| Rename in scope | ⌃⌘E |
| Stop building or running | ⌘. (Cmd Dot) |
| Indent left | ⌘[ |
| Indent right | ⌘] |
| Add API comments | ⌥⌘/ |
| Adjust indentations | ⌃I |
| Comment or uncomment | ⌘/ |
| Open quick help | ⌃⌘3 |
| Clear console | ⌘K |
