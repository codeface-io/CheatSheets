# Mac Cheat Sheet

## Keyboard Shortcuts

| Intent | Key Command |
| --------- | ----------- |
| Rename a selected file, folder or Numbers cell | ⌥⏎ |
| Delete a selected file or folder | ⌘⌫ |
| Show/hide hidden files | ⌘⇧. (Cmd Shift Dot) |

## Special Characters

| Character | Key Command |
| --------- | ----------- |
| ü         | ⌥u, u       |
| €         | ⌥⇧2         |
| –         | ⌥-          |
| —         | ⌥⇧-         |
| …         | ⌥:          |
| ß         | ⌥s          |
| •         | ⌥8          |
